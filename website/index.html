Get Started

VS Code / VS Code Insiders

Open in VS Code Open in VS Code Insiders

Or manually add to settings.json:

"chat.plugins.marketplaces": ["PlagueHO/plagueho.skills"]

GitHub Copilot CLI / Claude Code

Add the marketplace:

/plugin marketplace add PlagueHO/plagueho.skills

Then install a plugin:

/plugin install @plagueho-agent-skills

View available skills:

/skills
