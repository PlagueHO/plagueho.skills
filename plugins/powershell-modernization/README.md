# PowerShell Modernization Plugin

Modernization plugin that migrates PowerShell test suites from Pester v4 to
Pester v5, following the official migration guide and current Pester 5
conventions.

## Installation

```bash
# Using Copilot CLI
copilot plugin install powershell-modernization@plagueho-os
```

## What's Included

### Commands (Slash Commands)

| Command | Description |
|---------|-------------|
| `/powershell-modernization:upgrade-pester-v4-to-v5` | Reliably migrate PowerShell Pester v4 tests to Pester v5, applying the official v4→v5 migration guide and Pester 5 conventions. |
